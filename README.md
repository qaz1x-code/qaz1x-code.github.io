Привет! Я начинающий инженер кибербезопасности, увлекаюсь программированием, реверс-инжинирингом и разбираюсь как всё устроенно изнутри. Как хобби пишу плагины для Minecraft и настраиваю свой arch под себя.

Это мой личный сайт-портфолио — сделан с нуля на чистом HTML, без фреймворков и зависимостей.


## Что внутри

Сайт состоит из одного файла `index.html`. Никаких сборщиков и зависемостей просто один файл.

Из интересного в реализации:
- все иконки и аватарка записанны как base64 — сайт не зависит ни от внешних файлов
- элементы сайта появляются при скролле
- прогресс бары анимируются в момент попадания в зону видимости
- проекты открываются в отдельном окне с блюром от моего hyprland

Шрифты: **B612**, **Orbitron**
